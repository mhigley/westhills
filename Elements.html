Elements

Note: This is going to be a considerably condensed list of tags. I'll only be discussing some of the most commonly used tags.
For a complete list, please go to https://developer.mozilla.org/en-US/docs/Web/HTML/Element

TLDR:

  • <h1> - <h6>
  • <p>
  • <a>
  • <ul>
  • <img>

<h1> - <h6>

These are headline elements, <h1> being the highest section level and <h6> being the lowest.

"Importance" is a term we'll use often in lots of cases. An <h1> has greater importance than <h2>.

<h1>Heading One</h1>

<h2>Heading Two</h2>

<h3>Heading Three</h3>

<h4>Heading Four</h4>

<h5>Heading Five</h5>

<h6>Heading Six</h6>

<p>

This element represents a paragraph block.

Without identifying text with elements such as the paragraph tag, the browser has less information to understand your webpage.

Note: Choosing the proper tags to markup your webpage is called semantic HTML. Semantic HTML is the use of HTML markup to reinforce the meaning of the information in webpages.

<p>Your paragraph goes here.</p>

Do This:

  • Add two paragraphs to your elements.html page.

<a>

The anchor tag creates a link to other web pages, files, locations within the same page, email addresses, or any other URL across the internet.

This one element is the crux to the entire internet. The single most important tag of all.

The anchor tag has a very important requirement to work. It must include the href attribute. HTML element attributes provide important information. In this case, it is the destination for the user when they click the words "Link to Google".

<a href="https://google.com">Link to Google</a>

  • Direct them to wherever you want on the internet
  • How can we open a new tab when the user clicks the link? (Another element attribute)

<ul>

This element represents an unordered list of items, typically rendered as a bulleted list.

The list of items are marked up with <li> tags creating a parent/child relationship. If you want to use an <li>, it must be nested inside a <ul>. If you want to use the <ul>, the immediate child element must be an <li>.

This is a radically useful element for website layout as well as lots of things are lists!

<ul>

  <li>List Item One</li>

  <li>List Item Two</li>

  <li>List Item Three</li>

</ul>

  • Once you're done, figure out how to provide an order of importance to your movie picks.

<img>

This element embeds an image into the webpage.

Similarly to the <a> tag, the <img> requires certain element attributes to properly work. First is the src attribute. This pulls the URL of an image into the webpage when it loads. Also, image tags need an alt attribute. Although it is primarily meant for accessibility purposes, a webpage will not validate without the alternative text.

<img src="https://unsplash.it/500/500" alt="Image from Unsplash">
